La condition if est la plus utilisée des conditions

If

1 - If...

La conditon if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle va nous permettre d'exécuter un bloc de code si et Seulement si le résultat vaut true

Attention ! : exceptions

Tout test d'une condition va être exécutée dans un contexte booléen ; donc à true à l'exception des valeurs suivantes qui vont être évaluées à false

  • La valeur 0zéro
  • Le booléen false
  • une chaîne de caractères vide
  • la valeur null
  • la valeur undefined
  • la valeur NaN (Not a number)

Inverser la valeur logique d'un test

Le code placé dan le if n'est exécuté que si le résultat de la comparaison est true

Dans certaines situations , on préfèrevérifier si la comparaison est est false : on peut vérifier que c'est faux en comparant explicitaement le résultat de la comparaison à false
